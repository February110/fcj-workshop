### Week 1 Objectives:

* Set up and secure a personal AWS account for the internship lab environment.
* Understand IAM access control, AWS cost management, support resources, VPC networking, and EC2 compute basics.

### Week 1 Achievements:

* Created an AWS account and completed the initial setup, including billing alert and budget awareness.
* Understood IAM users, groups, roles, and policies, and practiced basic permission assignment.
* Learned how AWS Budgets and Support help control cost and handle account or service issues.
* Designed a basic VPC model with subnets, routing components, Internet Gateway, NAT Gateway, Flow Logs, and CloudWatch monitoring.
* Launched an EC2 instance and understood the role of AMI, Key Pair, EBS, security group, and instance state.
